The purpose of these user guidelines is to outline the functionality and use of the Procurement Enquiry Database.

This database is a system for recording the key details from each procurement, and attaching scanned copies of the full audit trail, as means of quickly being able to access historical procurement documentation for future reference - particularly during internal and external/donor audits.
