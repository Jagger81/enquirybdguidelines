The purpose of these user guidelines is to outline the functionality and use of the Procurement Enquiry Database.

This database is a system for recording the key details from each procurement, and attaching scanned copies of the full audit trail, as a means of quickly being able to access historical procurement documentation for future reference - particularly during internal and external/donor audits.

Understanding The Layout

Follow The Steps Below

1. Open the Enquiry Database and Enable Content

Enable Content


When you open the database you might be presented with a Security Warning message - if so, simply hit the Enable Content button

2. Main Menu

Main Menu Screenshot

The Main Menu shows four tabs;

Enquiry Options, Supplier Options, Search Options and Reporting Options

For now, the only one that is relevant (as highlighted above) is Enquiry Options. The other 3 tabs are under development and can be expanded on if/when the need arises.

The previous version of the database allowed for up to 5 "Activities" (i.e procurement items) to be entered under the same enquiry - this has since been decreased to 3, to allow for additional functionaility. If the user wishes to view older Enquiries in that format they can hit the button at the bottom right of the menu.

In the top right hand corner there is a link to the Intranet.
